Input description
#################
The ROOSTER input file is an ASCII text file consisting of lines (cards) identified by keywords. It should be named ``input``. 
The cards are presented in the following sections by examples in alphabetical order of the keywords. 
All numerical values and upper-case words are specified by users. 
The low-case words are keywords and should not be changed. The cards are divided in the following categories:

.. toctree::
    input-control
    input-core
    input-fluid
    input-solid
